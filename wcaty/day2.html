String Operation : Upper / Lower / Strip / Replace / Split

Upper, Lower

When we look at ASCII table, we can know that the difference between lowercase and uppercase is 32
65 A ~ 90 Z
97 a ~ 122 z
→ We can change the uppercase and lowercase by +32, -32 the ASCII code.
For calculation, we can either Make a new copy or just Replace it.
Replacing uses less bytes so it can be faster, but it is dangerous because we lose the original data.
(ex) "I'm lovin' it." =
[73, 39, 109, 32, 108, 111, 118, 105, 110, 39, 32, 105, 116, 46]





Strip

Striping is returning a copy of the string with both leading and trailing characters stripped.
For example, we can strip blank spaces from " Bob Grant" → "Bob Grant"
(SPACE = 32, TAB = 9, \n = 10, \c = 13)

Replace

For upper and lower, we could replace and edit the data because the total length of string doesn't change.
BUT, length of string changes when we replace.
(ex) "a message" : n=9 → "batman" : n=6 -3
Everytime we replace "a message" to "batman", the length of string gets shorter. (-3)*N
With this way, we can calculate the final string's length. len(Original)-3N

  1. Calculate the length of final string
  2. Copy the string with words replaced
  3. Delete the old data

Split

Spliting string is very useful when we get datas from files.
(ex)We want to make "Kyne, John, Grant, Alice" → ["Kyne", "John", "Grant", "Alice"]
We know that we can split "Kyne, John, Grant, Alice" with ", " to make ["Kyne", "John", "Grant", "Alice"]
We splited 3 time so we have 4 strings as a result.

What can be the worst case?

  • ", , , ..., , " → Spliting N times (N space)
  • "Onebigstringwithnospliting" → No spliting (N+1 space)

What about the big strings like a whole book? (ex) colour → color
Find Exact string matching at NIST DADS.
First, we have to match every strings to calculate the final length and then replace.
Brute force : O(mn)
Knuth-Morris-Pratt : O(m+n)

class Test {
  public static void main(String[] args) {

    Scanner kb = new Scanner(System.in);

    //Keyboard input and printf

    String name = kb.nextLine();
    System.out.println("How old are you?");
    int age = kb.nextInt();
    System.out.println("How tall are you?");
    double height = kb.nextDouble();

    System.out.printf("Your name is %s, you are %d years old, and you are %.2f meters tall.", name, age, height);

    //String split
    System.out.println("Enter numbers with ', '");
    String line = kb.nextLine();
    String[] parts = line.split(", ");
    int array_length = parts.length;

    for (int x = 0; x < array_length - 1 ; x++){
      System.out.print(parts[x] + " | ");
    }
    System.out.println(parts[array_length - 1]);

  }
}
