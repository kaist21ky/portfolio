Celebrity Problem

In a group of N people
Celebrity : Everyone knows celebrity but celebrity doesn'y know anyone else in the group.
Question : "Does A knows B?" → Yes / No

Q. What is the minimum number of questions to prove A is a celebrity?

If we ask "Does A knows B?", we can pick candidate celebrity.

If the answer is Yes : A is not a celebrity. (Celebrity should know nobody)
→ Change A
→ B is candidate celebrity.

If the answer is No : B is not a celebrity. (Everyone should know the celebrity)
→ Change B
→ A is candidate celebrity.

Repeat this until there is only one person left, who is the actual candidate celebrity. (Takes N-1 times)

Now, we can think of group with normal people and the candidate celebrity separately.
We know that this candidate is the only one who can be the celebrity, but it is not proved.
We need 2(N-1)-1 more questions to prove that this person is a real celebrity.

2(N-1)-1 + (N-1) = 3(N-1)-1 times

    int D_1 = kb.nextInt();
    System.out.print("Quiz #2 : ");

Geometry_Homework.java

Night 1 Geometry homework

import java.util.Scanner;

class Main {
  public static void main(String[] args) {
    System.out.println("===[ This program help you with your geometry homework ]===");
    System.out.println("  Mode 0 : The perimeter of a triangle");
    System.out.println("  Mode 1 : The circumference of a circle, from its radius");
    System.out.println("  Mode 2 : The circumference of a circle, from its diameter");
    System.out.println("  Mode 3 : The volume of a rectangular solid");
    System.out.println("  Mode 4 : The volume of a cylinder, from its height and radius");
    System.out.println("  Mode 5 : The volume of a sphere, from its radius");
    System.out.println("  Mode 9 : Exit");

    Scanner kb = new Scanner(System.in);

    while (true) {
      System.out.println("====================");
      System.out.print("MODE : ");
      int mode = kb.nextInt();

      if (mode == 0){
        System.out.println("Please enter 3 lengths of triangle --> Perimeter");
        double A_1 = kb.nextDouble();
        double A_2 = kb.nextDouble();
        double A_3 = kb.nextDouble();
        double A = A_1 + A_2 + A_3;
        System.out.println("Perimeter = " + A);
      } else if (mode == 1) {
        System.out.println("Please enter the radius --> Circumference");
        double B_1 = kb.nextDouble();
        double B = 2 * 3.14 * B_1;
        System.out.println("Circumference = " + B);
      } else if (mode == 2) { 
        System.out.println("Please enter the diameter --> Circumference");
        double C_1 = kb.nextDouble();
        double C = 3.14 * C_1;
        System.out.println("Circumference = " + C);
      } else if (mode == 3) {
        System.out.println("Please enter 3 lengths of rectangular solid --> Volume");
        double D_1 = kb.nextDouble();
        double D_2 = kb.nextDouble();
        double D_3 = kb.nextDouble();
        double D = D_1 * D_2 * D_3;
        System.out.println("Volume = " + D);
      } else if (mode == 4) {
        System.out.println("Please enter height and radius of cylinder --> Volume");
        double E_1 = kb.nextDouble();
        double E_2 = kb.nextDouble();
        double E = E_1 * 3.14 * E_2 * E_2;
        System.out.println("Volume = " + E);
      } else if (mode == 5) {
        System.out.println("Please enter the radius --> Volume");
        double F_1 = kb.nextDouble();
        double F = 4 * 3.14 * F_1 * F_1 * F_1 / 3;
        System.out.println("Volume = " + F);
      } else if (mode == 9) {
        break;
      }
    }
  }
}
